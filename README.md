# SageTools

#### 介绍

SageTools 一个简单的常用工具类和拓展方法包。

能少些重复代码就算成功哩~


#### 安装教程

.NET CLI

```
dotnet add package SageTools --version 2.0.1
```

Package Manager

```
NuGet\Install-Package SageTools -Version 2.0.1
```

PackageReference

```
<PackageReference Include="SageTools" Version="2.0.1" />
```
